## [1.0.0] - 2026-04-08

### 新增
- 🎉 项目初始化
- 📱 **即时通讯** - 私聊、群聊、表情回复、WebSocket 实时推送、已读回执
- 📊 **多维表格** - 表格/看板视图、字段自定义、拖拽排序、实时协作
- 📝 **云文档** - 富文本编辑器、AI 辅助写作、协作编辑、评论系统
- ✅ **任务管理** - 任务看板、状态跟踪、拖拽同步、优先级管理
- 🤖 **AI 助手** - 对话界面、流式输出、文本生成、代码助手
- 📁 **文件管理** - 文件夹管理、拖拽上传、文件预览、星标收藏
- 🎯 **OKR 系统** - 目标管理、关键结果、进度跟踪、周期管理
- 📅 **日历日程** - 月/周/日视图、事件管理、参与者管理、提醒功能
- 🏢 **企业管理** - 企业创建、成员管理、权限控制
- 🔐 **用户认证** - 注册、登录、JWT 认证、密码重置

### 技术实现
- ✅ Fastify 后端框架 + TypeScript
- ✅ Next.js 14 前端框架 + React 18
- ✅ PostgreSQL 数据库设计
- ✅ Redis 缓存/会话管理
- ✅ MinIO 对象存储
- ✅ WebSocket 实时通信
- ✅ Docker Compose 部署配置
- ✅ CI/CD GitHub Actions
