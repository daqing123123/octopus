## 五、功能免费/付费划分原则

### 永久免费（70%）
1. **核心协作功能** - 让团队能用起来
2. **基础AI能力** - 体验AI价值
3. **基础存储** - 满足日常使用
4. **基础视图** - 核心展示形式

### 暂时免费（30%）
1. **高级AI能力** - 后续可能按量收费
2. **高级自动化** - 后续可能订阅收费
3. **高级分析** - 后续可能订阅收费
4. **高级安全** - 后续可能企业收费

### 未来收费方向
1. **AI用量** - 超出配额后按token收费
2. **存储扩展** - 超出免费额度后收费
3. **企业高级功能** - 企业版订阅
4. **私有部署** - 一次性授权+年费
