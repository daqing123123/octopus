### 一、个人触手（Claw）- 私有AI助手

| 功能 | 描述 |
|------|------|

| **云文档** | 协作文档、实时编辑 |
| **多维表格** | 灵活的数据管理 |
| **任务管理** | 任务分配、进度跟踪 |
| **OKR管理** | 目标管理、进度评估 |
| **知识库** | 企业知识沉淀、搜索 |
| **AI助手** | 企业级AI对话 |
| **审批流** | 自定义审批流程 |

### 三、触手生命周期

| 阶段 | 功能 |
|------|------|
| **入职** | 档案创建、入职向导、任务清单 |

| **办公** | 日程协作、文档协作、任务完成 |
| **离职** | 交接清单、物品归还、数据保留 |
| **求职** | 简历管理、面试准备、工作机会 |

## 🏗️ 技术栈

### 后端
- **框架**: Fastify
- **数据库**: PostgreSQL
- **缓存**: Redis
- **向量数据库**: Qdrant
- **对象存储**: MinIO
- **实时通信**: Socket.io

### 前端
- **框架**: Next.js (App Router)
- **UI**: React + Tailwind CSS
- **状态管理**: Zustand
- **实时协作**: Yjs + ProseMirror

## 🚀 快速开始

### 环境要求
- Node.js >= 18
- PostgreSQL >= 14
- Redis >= 6
- Qdrant >= 1.0
- MinIO (可选)

### 安装

```bash
# 克隆项目
git clone https://github.com/daqing123123/octopus.git
cd octopus

# 安装后端依赖
cd src/backend
npm install

# 安装前端依赖
cd ../frontend
npm install

# 复制环境变量
cp .env.example .env
# 编辑 .env 填入配置

# 运行数据库迁移
cd ../..
npm run db:migrate

# 启动后端
npm run dev

# 启动前端（新终端）
cd src/frontend
npm run dev
```

### 访问
- 前端: http://localhost:3000
- API文档: http://localhost:3000/docs
